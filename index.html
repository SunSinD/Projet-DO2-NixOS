Le projet

Qu'est-ce que DO2 ?

Le projet DO2 — Don d'ordinateur, Deuxième vie — est une initiative du Département de technologie de génie électrique du Collège Montmorency. Les étudiants remettent en état des ordinateurs en fin de vie utile pour le collège, puis les distribuent gratuitement à des personnes dans le besoin, notamment des immigrants en francisation.

Chaque ordinateur est livré avec un écran, un clavier et une souris, et une version de Linux entièrement configurée en français, prête à l'emploi.

Pourquoi NixOS ?

NixOS est une distribution Linux unique : tout le système est décrit dans des fichiers de configuration. Cela signifie qu'une seule commande suffit pour reproduire exactement la même installation sur tous les ordinateurs — même configuration, mêmes applications, même langue.

C'est l'outil parfait pour un projet à grande échelle comme DO2, où l'objectif est de standardiser l'expérience sur une cinquantaine de machines.

Installation

Installer sur un ordinateur

  1. Booter depuis la clé USB NixOS
    Brancher la clé, allumer le laptop, appuyer sur F12 (ThinkPad) pour choisir la clé USB.
  2. Se connecter au Wi-Fi (si pas de câble Ethernet) :
    nmcli device wifi connect "NOM_WIFI" password "MOT_DE_PASSE"
  3. Lancer l'installation automatique :
    sudo bash <(curl -sL https://raw.githubusercontent.com/SunSinD/Projet-DO2-NixOS/main/install.sh)
    Le script détecte le disque, partitionne, installe et redémarre automatiquement. Durée : 10 à 30 minutes.

Identifiants par défaut — Utilisateur : user   Mot de passe : pass. La connexion automatique est activée ; l'utilisateur n'a pas besoin de se connecter au démarrage.
