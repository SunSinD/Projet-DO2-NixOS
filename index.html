Le Projet DO2 est une initiative du Département de technologie de génie électrique du Collège Montmorency. Nous remettons en état des ordinateurs en fin de vie utile pour le collège, puis les distribuons à des personnes dans le besoin, notamment des immigrants en francisation.

Chaque ordinateur portatif est remis en état et fourni à son futur utilisateur avec tout le nécessaire pour démarrer.

Complet à la sortie

Chaque portatif est remis à neuf et prêt à l'emploi dès la remise.

Protocole de déploiement

Installation automatisée

Le processus a été optimisé pour flasher une machine en moins de 15 minutes. Choisissez la méthode qui correspond à votre situation.

Tethering USB - Partager la connexion de votre téléphone
Méthode recommandée si vous êtes au Collège
1
Démarrer depuis la clé USB

Insérez la clé USB NixOS dans l'ordinateur. Allumez-le et appuyez sur F12 (ou Échap / F2 selon le modèle) pour ouvrir le menu de démarrage. Choisissez la clé USB dans la liste.

Hint : Si F12 ne marche pas, essayez F10, F11 ou Del. Sur certains ThinkPads, c'est le bouton Novo près du port d'alimentation.
Important : Dans le menu NixOS, choisissez la première option - NixOS LTS. Ne choisissez pas la version avec des chiffres.
2
Préparer votre téléphone

Votre téléphone doit être connecté au Wi-Fi Le_College_Montmorency et vous devez être connecté au portail du Collège (la page de connexion du Collège doit avoir été complétée dans votre navigateur). Branchez ensuite votre téléphone à l'ordinateur avec un câble USB.

Android : Paramètres - Connexions - Point d'accès mobile - Modem USB - Activer.
iPhone : Réglages - Partage de connexion - Autoriser les autres à rejoindre, puis brancher le câble.

Android (English): Settings → Connections → Mobile Hotspot and Tethering → USB Tethering → Enable.
iPhone (English): Settings → Personal Hotspot → Allow Others to Join, then plug in the cable.
3
Vérifier que la connexion USB est reconnue

Dans le terminal NixOS, tapez :

nmcli device status

Cherchez une interface comme usb0 ou enp0s20u1. Son état doit être connected.

4
Confirmer la connexion
ping -c 3 github.com

Attendez de voir des réponses avant de continuer. Si ça ne marche pas, vérifiez le tethering sur votre téléphone.

5
Lancer l'installation

Cette commande télécharge et lance le script d'installation depuis GitHub :

sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/SunSinD/Projet-DO2-NixOS/main/install.sh)"
Le script détecte le disque, partitionne, copie la configuration NixOS et installe le système. L'installation prend environ 10 à 30 minutes selon la vitesse du réseau. Ne fermez pas l'ordinateur pendant ce temps.
Wi-Fi Montmorency - Se connecter au réseau du Collège via le terminal
Utilisez cette méthode pour vous connecter au Wi-Fi du Collège Montmorency depuis le live USB NixOS

Ces commandes sont à exécuter dans le terminal NixOS, après avoir démarré sur la clé USB. Remplacez VOTRE_IDENTIFIANT et VOTRE_MOT_DE_PASSE par vos vraies informations du Collège avant de lancer les commandes.

1
Démarrer depuis la clé USB

Insérez la clé USB NixOS dans l'ordinateur. Allumez-le et appuyez sur F12 (ou Échap / F2 selon le modèle) pour ouvrir le menu de démarrage. Choisissez la clé USB dans la liste.

Hint : Si F12 ne marche pas, essayez F10, F11 ou Del. Sur certains ThinkPads, c'est le bouton Novo près du port d'alimentation.
Important : Dans le menu NixOS, choisissez la première option - NixOS LTS. Ne choisissez pas la version avec des chiffres.
2
Détecter automatiquement l'interface Wi-Fi
IFACE=$(nmcli -t -f DEVICE,TYPE device | awk -F: '$2=="wifi"{print $1; exit}')
Cette commande détecte automatiquement le nom de votre interface Wi-Fi (ex. wlan0, wlp3s0) et le sauvegarde dans la variable IFACE.
3
Créer le profil de connexion Wi-Fi
sudo nmcli con add type wifi con-name "Montmorency-WiFi" ifname "$IFACE" ssid "Le College Montmorency"
4
Configurer l'authentification WPA-EAP et se connecter

Remplacez VOTRE_IDENTIFIANT par votre numéro d'étudiant et VOTRE_MOT_DE_PASSE par votre mot de passe du Collège :

sudo nmcli connection modify "Montmorency-WiFi" \
  wifi-sec.key-mgmt wpa-eap \
  802-1x.eap peap \
  802-1x.phase2-auth mschapv2 \
  802-1x.identity "VOTRE_IDENTIFIANT" \
  802-1x.password "VOTRE_MOT_DE_PASSE"
sudo nmcli con up "Montmorency-WiFi"
⚠ À modifier avant d'exécuter :
— VOTRE_IDENTIFIANT → votre numéro d'étudiant du Collège (ex. 2477966)
— VOTRE_MOT_DE_PASSE → votre mot de passe du portail Omnivox
5
Vérifier la connexion
ping -c 3 github.com

Si vous recevez des réponses, la connexion est établie. Vous pouvez maintenant lancer l'installation.

6
Lancer l'installation

Cette commande télécharge et lance le script d'installation depuis GitHub :

sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/SunSinD/Projet-DO2-NixOS/main/install.sh)"
Le script détecte le disque, partitionne, copie la configuration NixOS et installe le système. L'installation prend environ 10 à 30 minutes selon la vitesse du réseau. Ne fermez pas l'ordinateur pendant ce temps.
Terminal - Wi-Fi à la maison ou ailleurs
Pour les installations hors du Collège, sur un réseau Wi-Fi normal
1
Démarrer depuis la clé USB

Insérez la clé USB NixOS dans l'ordinateur. Allumez-le et appuyez sur F12 (ou Échap / F2 selon le modèle) pour ouvrir le menu de démarrage. Choisissez la clé USB dans la liste.

Hint : Si F12 ne marche pas, essayez F10, F11 ou Del. Sur certains ThinkPads, c'est le bouton Novo près du port d'alimentation.
Important : Dans le menu NixOS, choisissez la première option - NixOS LTS. Ne choisissez pas la version avec des chiffres.
2
Se connecter au Wi-Fi

Si vous n'avez pas de câble Ethernet, connectez-vous à votre réseau Wi-Fi. Remplacez NOM_DU_WIFI et MOT_DE_PASSE par les informations de votre réseau :

nmcli device wifi connect "NOM_DU_WIFI" password "MOT_DE_PASSE"
Hint : Si vous avez un câble Ethernet, branchez-le directement - vous pouvez passer cette étape.
3
Vérifier la connexion
ping -c 3 github.com

Attendez de voir des réponses avant de continuer. Si ça ne marche pas, vérifiez votre connexion Wi-Fi.

4
Lancer l'installation

Cette commande télécharge et lance le script d'installation depuis GitHub :

Questions fréquentes

Vous avez des questions sur le Projet DO2 ? Voici les réponses aux questions les plus fréquentes, en simple et en français.

NixOS est un système Linux. Il est très stable, sécuritaire, et on peut le configurer de la même façon sur plusieurs ordinateurs en même temps. C'est parfait pour notre projet, car ça nous permet d'installer exactement la même chose sur toutes les machines d'un seul coup.

Les ordinateurs sont destinés aux étudiants du programme de francisation du Collège Montmorency. Les appareils sont attribués directement par le Collège lors des séances de remise organisées par notre équipe.

Non, pas du tout. L'ordinateur vient avec une interface simple et en français. On a choisi un bureau facile à utiliser, similaire à ce que vous connaissez peut-être déjà. Un guide d'utilisation est fourni avec l'ordinateur pour vous aider à démarrer.

Chaque ordinateur inclut une suite bureautique complète, un navigateur web, un lecteur multimédia et d'autres outils du quotidien. Tout est préinstallé et configuré pour une utilisation immédiate dès la remise.

Le Projet est géré par les étudiants du programme de génie électrique du Collège Montmorency. Le code source est disponible sur GitHub et les contributions techniques sont les bienvenues. Vous pouvez aussi contacter le Département de génie électrique pour en savoir plus.

Vous pouvez écrire un message sur la page GitHub du projet ou contacter directement le Collège Montmorency. L'équipe d'étudiants fait de son mieux pour aider, surtout lors des séances de distribution où une courte formation est aussi offerte.
