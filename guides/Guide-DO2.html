Se connecter au Wi-Fi

  1. Regardez en bas à droite de votre écran.
  2. Cliquez sur l'icône de réseau (les petites barres).
  3. Choisissez votre réseau Wi-Fi dans la liste.
  4. Entrez le mot de passe du Wi-Fi et cliquez "Connecter".

Changer votre nom et mot de passe

Le nom d'utilisateur par défaut est "Utilisateur" et le mot de passe par défaut est "pass".

  1. Cliquez sur le bouton "Menu" en bas à gauche.
  2. Tapez "Utilisateurs" dans la barre de recherche.
  3. Cliquez sur "Utilisateurs et groupes".
  4. Cliquez sur "Utilisateur" dans la liste.
  5. Pour changer le nom : cliquez sur "Utilisateur" et tapez le nouveau nom.
  6. Pour changer le mot de passe : cliquez sur "Mot de passe", entrez "pass" comme ancien mot de passe, puis tapez votre nouveau mot de passe et confirmez.
Astuce : Le mot de passe actuel est "pass". Nous vous recommandons de le changer.

Installer de nouvelles applications

  1. Cliquez sur le bouton "Menu" en bas à gauche.
  2. Cherchez "Logithèque" (ou "Software").
  3. Parcourez ou cherchez l'application souhaitée.
  4. Cliquez sur "Installer".
Astuce : Toutes les applications de la Logithèque sont gratuites et sécuritaires.

Utiliser le traducteur

  1. Ouvrez Dialect depuis le menu.
  2. Choisissez la langue source (votre langue) à gauche.
  3. Choisissez Français à droite.
  4. Tapez ou collez le texte à traduire.
  5. La traduction apparaît automatiquement!

Utiliser Microsoft Office en ligne

Vous avez LibreOffice sur votre bureau (Writer = Word, Calc = Excel). Si vous avez besoin de Microsoft Office :

  1. Ouvrez Google Chrome.
  2. Allez à office.com.
  3. Connectez-vous avec votre compte Microsoft.

Guide LibreOffice

LibreOffice est la suite bureautique installée sur votre ordinateur. Voici les applications :

  • Writer — Traitement de texte (comme Microsoft Word). Pour écrire des lettres, rapports, CV.
  • Calc — Tableur (comme Microsoft Excel). Pour faire des calculs, tableaux, graphiques.
  • Impress — Présentations (comme Microsoft PowerPoint). Pour créer des diaporamas.
  • Draw — Dessin et schémas. Pour créer des affiches, des plans.
  • Math — Éditeur de formules mathématiques.
  • Base — Base de données (comme Microsoft Access).

Pour ouvrir une application LibreOffice, cliquez sur le bouton "Menu" en bas à gauche et cherchez son nom.

Astuce : Les fichiers Word (.docx), Excel (.xlsx) et PowerPoint (.pptx) s'ouvrent directement dans LibreOffice.

Redémarrer l'ordinateur

  1. Cliquez sur le bouton "Menu" en bas à gauche.
  2. Cliquez sur l'icône d'arrêt (en bas du menu).
  3. Choisissez "Redémarrer" ou "Éteindre".
Important : Si quelque chose ne fonctionne pas, essayez de redémarrer l'ordinateur. Cela règle la plupart des problèmes!
