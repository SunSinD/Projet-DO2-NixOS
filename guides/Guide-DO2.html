1. Navigation

Voir la vidéo

Ouvrir l'ordinateur

L'ordinateur affiche un écran de connexion au démarrage.

  1. Entrez le mot de passe : pass
  2. Appuyez sur Entrée.
  3. Le bureau apparaît avec le logo DO2 au centre.
Mot de passe par défaut : pass

Le Menu

Le Menu donne accès à tous les programmes de l'ordinateur.

  1. Cliquez sur Menu en bas à gauche de l'écran.
  2. Le menu des applications s'ouvre.
  3. À gauche : les catégories (Bureautique, Graphisme, Internet...).
  4. À droite : la liste des programmes disponibles.
  5. Cliquez sur un programme pour l'ouvrir.

Bureautique

La catégorie Bureautique contient les outils pour écrire et calculer.

Astuce : Les fichiers .docx, .xlsx et .pptx s'ouvrent directement dans LibreOffice.

Ouvrir les Paramètres système

Pour ouvrir les Paramètres système :

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, cliquez sur l'icône Paramètres système.
  3. La fenêtre s'ouvre avec quatre sections : Apparence, Préférences, Matériel, Administration.

Ouvrir le Gestionnaire de fichiers

Pour ouvrir le Gestionnaire de fichiers :

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, juste en dessous de Paramètres système, cliquez sur l'icône Gestionnaire de fichiers.
  3. Le Dossier personnel s'ouvre.
  4. À gauche : raccourcis vers vos dossiers principaux.
  5. Au centre : le contenu du dossier sélectionné.

Verrouiller l'écran

Pour verrouiller l'écran sans éteindre l'ordinateur :

  1. Cliquez sur Menu en bas à gauche.
  2. Cliquez sur l'icône de verrou (cadenas) en bas du menu.
  3. L'écran se verrouille. Entrez votre mot de passe pour revenir.

Déconnecter l'utilisateur

Pour changer d'utilisateur sans éteindre l'ordinateur :

  1. Cliquez sur Menu en bas à gauche.
  2. Cliquez sur l'icône de déconnexion (flèche de sortie) en bas du menu.
  3. Confirmez la déconnexion.
  4. L'écran de connexion réapparaît.

Éteindre l'ordinateur

  1. Cliquez sur Menu en bas à gauche.
  2. Cliquez sur l'icône d'arrêt (cercle avec une ligne).
  3. Choisissez Éteindre pour fermer complètement, ou Redémarrer pour relancer.
Important : Si quelque chose ne fonctionne pas, essayez de redémarrer l'ordinateur. Cela règle la plupart des problèmes!

2. Périphériques physiques

Voir la vidéo

Les touches spéciales du clavier

Voici les principales touches spéciales :

  • Ctrl + C : copier
  • Ctrl + V : coller
  • Ctrl + Z : annuler la dernière action
  • Ctrl + H : afficher les fichiers cachés (dans le Gestionnaire de fichiers)
  • F9 : afficher ou cacher le panneau latéral
  • Alt + Tab : passer d'une fenêtre ouverte à l'autre

Bouton +/-

Le bouton +/- permet de régler le volume directement sur l'appareil.

  • + : augmenter le volume
  • - : baisser le volume

Connecteurs (USB, HDMI, Carte SD, Audio jack, Chargeur)

L'ordinateur possède plusieurs ports de connexion :

  • USB : brancher une clé USB, souris, clavier, etc.
  • HDMI : connecter un écran ou un projecteur externe
  • Carte SD : insérer une carte mémoire (appareil photo, etc.)
  • Audio jack (3,5 mm) : brancher des écouteurs ou un micro
  • Chargeur : recharger la batterie de l'ordinateur

3. Google

Voir la vidéo

Google Chrome est le navigateur principal installé sur votre ordinateur DO2.

Utiliser Google Chrome

  1. Cliquez sur l'icône Google Chrome dans la barre en bas ou dans le Menu.
  2. La page d'accueil s'ouvre.
  3. Tapez une adresse web ou une recherche dans la barre en haut.

Utiliser Microsoft Office en ligne

  1. Ouvrez Google Chrome.
  2. Allez à office.com.
  3. Connectez-vous avec votre compte Microsoft.

Utiliser le Traducteur

  1. Ouvrez Dialect depuis le Menu.
  2. Choisissez la langue source (votre langue) à gauche.
  3. Choisissez Français à droite.
  4. Tapez ou collez le texte à traduire.
  5. La traduction apparaît automatiquement!

4. Apparence et Personnalisation

Voir la vidéo

La section Apparence permet de changer le look du bureau. Elle contient quatre options :

  • Arrière-plans : choisir l'image du fond d'écran
  • Choix de police : changer la taille et le style des lettres
  • Effets : activer ou désactiver les animations des fenêtres
  • Thèmes : changer le style global (couleurs, icônes, curseur)

Changer le fond d'écran

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, cliquez sur Paramètres système.
  3. Dans la section Apparence, cliquez sur Arrière-plans.
  4. Choisissez un dossier d'images à gauche.
  5. Cliquez sur une image pour l'appliquer.
  6. Dans l'onglet Paramètres, vous pouvez choisir comment l'image est affichée : Centré, Étiré, Zoom, etc.

Personnalisation du panneau en bas

Le panneau du bas (barre des tâches) peut être personnalisé :

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, cliquez sur Paramètres système.
  3. Allez dans Préférences, puis cliquez sur Panneau.
  4. Vous pouvez changer la hauteur, la visibilité, et la taille des icônes.
  5. Cliquez sur Mode de modification du panneau pour déplacer les éléments.
Astuce : La barre en bas contient : Menu, Google Chrome, Gestionnaire de fichiers, Outlook, Timeshift. À droite : Bluetooth, volume, Wi-Fi et horloge.

Applications sur le bureau

Pour ajouter une application sur le bureau :

  1. Cliquez sur Menu en bas à gauche.
  2. Trouvez l'application souhaitée.
  3. Faites un clic droit sur l'application.
  4. Choisissez Ajouter au bureau.

Pour retirer une icône du bureau :

  1. Faites un clic droit sur l'icône sur le bureau.
  2. Choisissez Supprimer.

5. Préférences

Voir la vidéo

La section Préférences est la plus grande. Elle permet de configurer le comportement du bureau. Voici les options les plus utiles :

  • Accessibilité : aides visuelles, grand texte, lecteur d'écran, clavier virtuel
  • Bureau : choisir les icônes affichées sur le bureau (Dossier personnel, Corbeille...)
  • Date et heure : régler le fuseau horaire et le format de l'heure
  • Notifications : activer ou désactiver les messages qui apparaissent en haut à droite
  • Panneau : configurer la barre en bas de l'écran
  • Fenêtres : contrôler le comportement des fenêtres
Astuce : Pour un usage de tous les jours, les réglages par défaut fonctionnent très bien. Si vous ne comprenez pas un réglage, laissez-le tel quel.

Matériel

La section Matériel permet de régler les équipements de l'ordinateur :

  • Affichage : résolution de l'écran et taux de rafraîchissement
  • Bluetooth : connecter des appareils sans fil (casque, clavier...)
  • Clavier : vitesse de frappe, raccourcis, langue
  • Gestion de Courant : régler quand l'écran s'éteint et le mode d'économie d'énergie
  • Son : régler le volume, la balance et tester les haut-parleurs
  • Souris : régler la vitesse du pointeur et le double-clic
Astuce : Si vous avez un problème avec le son, l'écran ou le clavier, c'est dans cette section qu'il faut chercher.

6. Administration

Voir la vidéo

La section Administration permet de gérer les comptes utilisateurs.

Ajouter un nouveau compte

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, cliquez sur Paramètres système.
  3. Dans la section Administration, cliquez sur Utilisateurs et groupes.
  4. Entrez votre mot de passe quand l'ordinateur vous le demande.
  5. Cliquez sur Add (Ajouter).
  6. Entrez le nom complet de la personne. Le nom d'utilisateur se crée automatiquement.
  7. Cliquez sur Ajouter.
  8. Créez un mot de passe pour ce compte (choisissez Moyen ou mieux).
  9. Cliquez sur Changer.
Important : Le compte par défaut s'appelle "Utilisateur" avec le mot de passe "pass". Changez-le pour plus de sécurité.

Gestionnaire de fichiers

Le Gestionnaire de fichiers permet d'organiser tous vos documents.

  1. Cliquez sur Menu en bas à gauche.
  2. En haut à droite du menu, juste en dessous de Paramètres système, cliquez sur Gestionnaire de fichiers.

Dossier personnel

Le Dossier personnel contient tous vos fichiers et dossiers :

  • Documents : vos fichiers texte, PDF, etc.
  • Downloads : fichiers téléchargés depuis Internet
  • Music : fichiers audio
  • Pictures : vos photos et images
  • Videos : vos vidéos

En bas de la fenêtre, vous voyez le nombre de fichiers et l'espace disponible sur le disque.

Bureau

Les icônes du bureau sont aussi accessibles dans le Gestionnaire de fichiers :

  1. Ouvrez le Gestionnaire de fichiers.
  2. Cliquez sur Bureau dans le panneau à gauche.
  3. Vous voyez les mêmes icônes que sur votre bureau : Guide DO2, LibreOffice Calc, LibreOffice Writer.

Fichiers récents

Pour retrouver rapidement vos derniers fichiers ouverts :

  1. Ouvrez le Gestionnaire de fichiers.
  2. Cliquez sur Fichiers récents dans le panneau à gauche.
  3. La liste de vos fichiers récemment utilisés apparaît.

Corbeille

La Corbeille contient les fichiers que vous avez supprimés.

  • Pour restaurer un fichier : cliquez droit dessus, puis cliquez sur "Restaurer".
  • Pour supprimer pour toujours : cliquez droit, puis "Supprimer définitivement".
  • Pour vider la corbeille : cliquez sur "Vider la corbeille".

Barre d'outils du Gestionnaire

En haut de la fenêtre du Gestionnaire de fichiers, on trouve les menus :

  • Fichier : créer un nouveau dossier, un nouvel onglet, ou fermer la fenêtre
  • Affichage : changer l'apparence des fichiers (icônes, liste, compact), afficher les fichiers cachés (Ctrl+H)
  • Aller à : naviguer directement vers un dossier (Bureau, Documents, Téléchargements...)
  • Aide : accéder à l'aide et voir les raccourcis clavier (Ctrl+F1)
